# 💊 Pharma Compliance Analytics Dashboard

An interactive Pharma Compliance Analytics Dashboard built using **Python, Streamlit, Pandas, and Plotly**. This project helps analyze pharmaceutical compliance data using interactive filters, KPI cards, charts, and downloadable reports.

---

## 📌 Features

- Interactive dashboard built with Streamlit
- KPI Cards
  - Total Compliance Cases
  - Average Risk Score
  - Critical Cases
  - Open CAPA
  - Closed CAPA
- Sidebar filters
  - Company
  - Department
  - Plant
  - Severity
- Interactive charts
  - Department-wise Compliance Cases
  - Severity Distribution
  - CAPA Status
  - Root Cause Analysis
  - Plant-wise Compliance Cases
  - Product-wise Compliance Cases
  - Monthly Compliance Trend
- Download report in Microsoft Word format

## ▶️ Run the Project

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the dashboard:

```bash
streamlit run dashboard.py
```
